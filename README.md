# HR Analytics Dashboard

For this project, I decided to apply some of the knowledge I’ve gained so far in Generation’s data analysis classes. I found a data table with fictional information and used several functions on it, such as:
- sum()
- average() and averageif()
- filter()
- count() and countif()

I also took the opportunity to practice generating possible charts with the information from this table, in order to highlight the most relevant data within it.
